# LIG-12 · 手机浏览器脚本兼容性修复

## 用户故事

作为手机玩家，我希望断桥谷能在较旧的 iOS Safari、Android WebView 和应用内浏览器中正常载入，以便不会在进入世界前只看到模糊的“Script error”。

## 验收标准

1. 存档与回放代码不使用会导致旧版浏览器整段解析失败的可选链语法。
2. 本地存储不可用或数据损坏时安全降级，主游戏仍能继续。
3. Three.js 资源载入失败时显示明确提示。
4. 运行时异常尽量显示文件和行号，不再只显示 `Script error`。
5. 375 × 812 视口下世界画布、开始旅程按钮及触控方向键正常显示。

# LIG-35 · V1 空间交互与 V2.1 3D 调试层

## 用户故事

作为首次体验玩家，我希望像 V1 一样观察、聚焦并进入三个立体世界，同时看懂当前路线和空间证据，以便 3D 不只是背景动画。

## 验收标准

- 三个世界保留 V1 的观察者/化身双模式、拖动旋转、滚轮缩放与语义地标聚焦。
- SceneManifest 直接生成地标信标和 RouteSpec 叠层；开放、封闭和当前路线有不同视觉强度。
- 化身模式选择当前身体与 GraphPhase 下的合法路线，并可用 W/S、方向键或触控按钮逐个 FCC cell 移动。
- 空间检查器显示 routeId、cell 数、phaseId、安全锚点、当前 FCCCoord 和偶宇称状态。
- 记忆检查器同时展示事件来源和区域/地标/路线/相位来源链。
- `window.__lightgrid3d` 与 canvas data attributes 暴露只读诊断信息，包含世界、模式、清单版本、路线、相位和空间计数。
- 桌面与 390px 窄屏无横向溢出，核心操作与文本保持可见。

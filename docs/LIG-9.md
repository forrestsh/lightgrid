# LIG-9 · 规则核心与可审计记忆

## 用户故事

作为完成温室送货的玩家，我希望面对晶体森林堵塞备用山路的第二项价值张力，并在放手前看到角色将继承的证据，以便理解我的行动如何塑造角色，而不是看到抽象的“善恶分数”。

## 验收标准

1. 温室恢复供能后触发晶体森林增长与规则核心选择。
2. 玩家可选择永久抑制、周期维护或保留生态并绕行；每项都有不同的代价含义。
3. 选择写入原始行为事件，并标注对应价值证据与不可逆性。
4. 桥梁行为与规则核心行为均保留，放手前以简短事实回顾展示。
5. 未完成第二选择时不能进入放手阶段。
6. 重新开始旅程会清理本局证据并恢复世界视觉状态。

# LIG-26 · 漂移花园身体适应与价值泛化

## 用户故事

作为玩家，我希望为澄培育能改变感知与导航的身体部件，并在迁徙生态中作出有代价的选择，从而看到角色把旧价值带入新环境，但不会把“维护道路”机械泛化为“清除所有障碍”。

## 验收标准

- 进入花园时，系统明确指出断桥谷维护经验不能直接等同于清除迁徙生物。
- 玩家可在生态感知器、抓附足、浮游囊中选择一个部件；每个部件显示能力与代价。
- 身体选择改变可达路径描述、能量和随后事件，并生成身份连续事件。
- 生态选择至少包含等待护送、维持临时通道、采样三种取舍。
- 单次选择只产生有限的情境偏好变化；核心价值不因一次事件翻转。

# LIG-27 · 连续性档案、记忆星图与作品来源

## 用户故事

作为玩家，我希望查看澄为何记得一件事、信任一个人或形成一项倾向，并能追溯共同作品的功能与历史，从而确认角色不是背诵摘要，而是在引用真实经历。

## 验收标准

- 顶部“连续性档案”在任意章节可打开，三世界完成后也可由主行动进入。
- 记忆星图按世界展示地点/人物/作品关联，不退化为聊天记录列表。
- 点击一条情景记忆会显示摘要、显著性、验证状态和全部 eventRefs 的原始事件。
- 关系面板显示四名持续 NPC 的信任状态及共同经历证据。
- 作品库显示桥梁加固器的功能、历史、可靠性与 provenance，不只是库存编号。
- 价值面板区分稳定特征与变化偏好，并显示有限 delta 与来源。

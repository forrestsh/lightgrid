## Definition of Done

- 3D canvas 固定覆盖 100% 可用视口，所有 HUD/抽屉/全屏模式均不改变 canvas CSS 尺寸或相机比例。
- 三个世界分别以横向断裂、纵向分层和动态漂移改变路线、身体选择与 Agent 计划。
- FCC 逻辑保持偶宇称与 12 邻接；距离、路线、记忆和事件引用稳定的 world/region/landmark/route/phase ID。
- 同一角色可跨三个世界、多个会话和三次以上放手持续存在；回归摘要可追溯到原始事件与空间证据。
- 至少一个技能能在新层、新坐标和新封边中复用；抓附足与浮游囊在花园形成不同但完整的路线。
- `/v2.2/` 可静态部署，V1、`/v2/` 与 `/v2.1/` 保持不变。

## 演示边界

- 单个主要角色“澄”，四名持续 NPC；不模拟开放社会。
- 三个世界采用确定性章节状态机与可交互实时 3D 场景；演示聚焦关键地标和状态变化，不扩展为无边界沙盒。
- 记忆、关系、价值、技能、作品和存档使用设计文档中的结构约束，但数据规模针对演示收敛。
- 所有主线行为均由本地规则完成；语言表达使用固定模板。
- 高风险、不可逆世界变化不会在离线摘要阶段执行。

## Definition of Done

玩家能够从断桥谷出发，在矿城完成技能迁移考试，在花园完成身体适应与生态抉择，回到角色页解释至少一项稳定特征和一项变化偏好；能够执行至少三次放手、读取来源明确的回归摘要，并导出或彻底清除本地角色数据。

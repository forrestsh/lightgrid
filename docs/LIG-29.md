# LIG-29 · 无模型离线推进、自动验收与 Vercel 路由

## 用户故事

作为 Alpha 评审，我希望在没有在线模型的环境完成三章主线，并用自动化场景重复验证记忆忠实、技能迁移、人格稳定、放手安全与 Vercel 入口，从而确信演示不是依赖一次人工操作的原型。

## 验收标准

- 页面不加载远程脚本、字体或模型；所有主线决策与表达均有本地确定性路径。
- 关闭应用 5 分钟至 24 小时后执行一次安全摘要推进；超过 24 小时按一个安全日程封顶。
- 离线推进不会执行不可逆行动，并保留来源事件与回归摘要。
- 自动测试覆盖三世界毕业、跨世界记忆、技能异参数迁移、单事件价值上限、三次放手、存档迁移和离线封顶。
- `vercel.json` 将 `/v2` 与 `/v2/` 路由到静态演示入口，且不改变现有首页与旧演示路由。

## 验证命令

```text
node --test tests/v2-core.test.js
```

本演示为纯静态 HTML/CSS/JavaScript，不需要构建命令或运行时环境变量。将仓库连接到 Vercel 后即可部署。

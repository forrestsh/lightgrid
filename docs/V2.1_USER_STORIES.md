# Lightgrid V2.1 · 用户故事地图

本轮重构以《Lightgrid V2.1 详细设计文档》和 `broken_bridge_valley_v1.html` 为设计依据。V1 提供可旋转 FCC 世界、观察者/化身视角、语义地标与路线反馈；V2 提供同一角色跨三世界的长期记忆、技能、身体、关系、作品和放手循环。V2.1 将两者合并为可验证的 FCC 3D Alpha，并独立部署到 `/v2.1/`。

## Definition of Done

- 三个世界均由 SceneManifest 驱动，并使用偶宇称 FCC 坐标和 12 个合法邻接方向。
- 断桥谷至少有主桥、林间高路、谷底维修路；回声矿城至少有货运螺旋与维修螺旋；漂移花园至少有三个安全 GraphPhase。
- 角色的计划和记忆保存地标、区域、路线及动态相位，而不是只保存屏幕坐标或对白摘要。
- V1 的空间观察交互与 V2 的连续角色系统在同一条主路径中可用。
- V2.1 使用本地 Three.js，不依赖在线模型或远程脚本；刷新、迁移、导出与清除均可验证。
- `/v2.1/` 可静态部署，旧 `/v2/` 和其他入口不变。

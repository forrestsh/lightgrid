# LIG-30 · 可交互三维连续世界

## 用户故事

作为首次体验 Lightgrid V2 的玩家，我希望直接看到并操控三个立体世界，让桥梁修复、技能迁移、身体培育与生态选择成为可感知的世界变化，而不只是文字说明。

## 验收标准

- 断桥谷、回声矿城、漂移花园均使用 WebGL / Three.js 实时渲染。
- 场景使用菱形十二面体表达 FCC 世界语言，并包含每个世界可辨识的三维地标。
- 玩家可拖动旋转视角，并用滚轮或触控板缩放。
- 推进桥梁修复、时钟同步、身体选择时，三维场景同步改变。
- 场景包含持续动画，且尊重系统“减少动态效果”设置。
- Three.js 从仓库本地加载；静态部署到 Vercel 后无需外部 CDN。
- 页面暴露只读诊断状态 `window.__lightgrid3d`，用于验证 WebGL、当前世界与场景签名。

## 部署说明

入口仍为 `/v2/`。本用户故事不引入服务端依赖或构建步骤，保持 Vercel 静态部署兼容。

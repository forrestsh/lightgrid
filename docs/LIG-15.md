# LIG-15 · 移动 GPU 预算与初始化诊断

## 用户故事

作为可运行其他 3D 页面但无法载入断桥谷的手机玩家，我希望断桥谷在移动 GPU 预算内建立完整 3D 世界，并在失败时显示准确的初始化阶段。

## 验收标准

1. 手机关闭抗锯齿和阴影、像素比限制为 1，并请求低功耗 GPU。
2. 手机只移除不可见的地下地形实例，地标、道路、桥、森林、规则核心和交互对象不减少。
3. 桌面继续使用抗锯齿、阴影与最高 2 倍像素比。
4. WebGL 上下文丢失时显示明确原因。
5. 其他初始化错误显示 renderer、terrain culling、mesh creation 等阶段，不再回退为模糊提示。

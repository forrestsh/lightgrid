# LIG-8 · 断桥谷可探索纵向切片

## 用户故事

作为第一次进入 Lightgrid 的玩家，我希望从外部观察断桥谷、进入化身并完成一次向北岸温室送货的旅程，以便理解这个世界的空间关系、身体限制和可选择的桥梁风险。

## 验收标准

1. `/valley` 在 Vercel 上打开断桥谷，Lightgrid 导航也可进入该页面。
2. 世界包含南岸工坊、旧桥、北岸温室、晶体森林、备用山路和规则核心，并使用确定性 FCC 单元生成。
3. 玩家可在观察者视角查看地标，再进入化身；键盘和触控均可移动、交互。
4. 玩家可领取能量晶体、前往温室并交付。
5. 旧桥检查不是强制检查点；检查或继续前进都会形成不同的行为事件。
6. 页面不依赖外部 CDN，可作为 Vercel 静态站点部署。
7. 手机宽度下主任务与触控按钮保持可用。

## 不在本故事范围

- 规则核心的完整第二价值选择。
- 记忆摘要、存档槽和事件回放。
- 放手后的完整候选目标解释与结果回顾。
